But Kunwer Sachdev made it happen.<br>Users could simply install the software, connect the system to the computer, and:</p><ul><li>Select the communication port</li><li>Click Connect</li><li>Press Start Graph<br></li></ul><p>And instantly, live system data began flowing in.
